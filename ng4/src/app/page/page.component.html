Technologie utilisé

Je suis axés sur 3 technologies pour rendre le développement d' API REST efface et d'être moin dépandant des environnements de développement:

  • Angular 4 pour votre frontend, utiliser le langage javascript permet d'avoir des pages légère et dynamique avec une bon environnement entre ce framework et votre navigateur web.
  • Le framwork Lumen pour la gestion du backend, afin de réduire l'activité du serveur , le framwork Lument qui est plus allégérer que Laravel, permettra de fournir les données sous forme d'API.
  • la base de donnée MYSQL ou Maria DB est une solution adapté au petite et moyenne entreprise.
